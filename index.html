Questions

Frequently asked

Everything you need to know about key conversion and security.

Is it safe to paste my private key here?
Yes. All conversion happens entirely in your browser using client-side JavaScript. Your private key is never transmitted to any server, stored in any database, or logged anywhere. There is no backend — the tool is a static web page. For maximum security, you can even disconnect from the internet after the page loads.
What's the difference between JSON and Base58 formats?
JSON format represents your 64-byte keypair as a numeric array like [174, 47, 154, ...]. It's used by the Solana CLI and older wallet versions. Base58 is a compact string encoding using the Solana Base58 alphabet, preferred by modern wallets like Phantom. Both represent the same underlying key data.
Can I use the converted key in Phantom wallet?
Yes. Phantom prefers Base58 format for private key imports. If you have a JSON array key, convert it to Base58 using this tool, then paste it into Phantom's "Import Private Key" field. If you see a "deprecated format" error in Phantom, it means you're trying to import JSON — convert to Base58 first.
How does the Solscan button work?
When you convert a 64-byte Solana keypair, the tool extracts the public key (the last 32 bytes) and Base58-encodes it to derive your wallet address. The "View on Solscan" button opens solscan.io/account/<address> in a new tab so you can inspect the wallet's transaction history, token holdings, and on-chain activity. This derivation happens entirely client-side — no server call is made.
What happens to my data after conversion?
Nothing is stored. The input and output exist only in your browser's memory while the page is open. When you close the tab or navigate away, everything is gone. We don't use cookies, localStorage, or sessionStorage. Refresh the page and it's a clean slate.
Can I verify the conversion is correct?
Absolutely. Convert your key to Base58, then convert it back to JSON — the result will match your original input byte-for-byte. The conversion is a lossless bidirectional transformation. You can also verify the derived public key matches across formats, or check the wallet on Solscan to confirm the address is correct.
Does this work offline?
Yes. Once the page is loaded, all conversion and address derivation works without any internet connection. The only feature that requires internet is the Solscan link itself, since it opens an external website. For the highest security, load the page, disconnect from the internet, then paste and convert your keys.
